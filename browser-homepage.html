Setting your browser's home page

This page is intended to provide you with a basic how-to on setting the home page of Chrome, Firefox, or Internet Explorer. Using Safari? Click here.

Also included are links to resetting the browser to clear history and manage privacy settings in the most common browsers. This is something you may wish to do on a regular basis, both to protect yourself and to give your browser optimal speed.

Chrome

Follow these steps to adjust your homepage settings:

  1. Click the Chrome menu in the upper corner of the browser toolbar. (Depending on what version of Chrome you are using, you may see a wrench icon)
  2. Select Settings
  3. Continue either to setting the browser startup page or setting the browser home button
Setting the browser startup page
  1. Within the Settings page, locate the On Startup section
  2. Choose the Open a specific page or set of pages option
  3. Click the Set page link to set the home page or add multiple startup pages
  5. Enter the URL you wish to add: www.eureka.edu/faculty
  7. Click OK
Setting the browser Home button
  1. Within the Settings page, locate the Appearance section
  2. Check Show Home button
  3. Click the Change link to set
  5. Enter the URL you wish to add: www.eureka.edu/faculty
  7. Click OK

That's it! Close and reopen the browser to see the changes.

Firefox

Setting the browser Home page
  1. Open the website you want to set as your home page (www.eureka.edu/faculty)
  2. Click the icon to the left of the web address, drag it to the Home button, and release
  3. Click Yes to set this as your new home page
Modifying the browser Home page
  1. On Windows, open the Options menu as shown below. On a Mac, select Firefox and then Preferences
  3. Set When Firefox starts: to Show my home page
  4. In the Home Page input, enter the URL you wish to add: www.eureka.edu/faculty
  6. Click OK

That's it! Close and reopen the browser to see the changes.

Internet Explorer

Setting the browser Home page
  1. In the upper corner, click Tools and select Internet Options
  3. Under the General tab, enter the web address of your new home page (www.eureka.edu/faculty)
  5. Choose Apply

That's it! Close and reopen the browser to see the changes.
